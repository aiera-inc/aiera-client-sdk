useAutoScroll

  • useAutoScroll<E, T>(opts?: { skip?: boolean; pauseOnUserScroll?: boolean; initialBehavior?: ScrollIntoViewOptions["behavior"]; behavior?: ScrollIntoViewOptions["behavior"]; block?: ScrollIntoViewOptions["block"]; inline?: ScrollIntoViewOptions["inline"] }): [RefCallback<E>, RefCallback<T>]
    • Defined in src/lib/hooks/useAutoScroll/index.ts:19

    Implementation of autoscroll that watches a target and autoscrolls anything the target changes. Target can be a targetRef that you want to keep in view or it can be simply the length of an array that changes and causes a scroll to the bottom.

    Type parameters

    • E: HTMLElement = HTMLDivElement

    • T: HTMLElement = HTMLDivElement

    Parameters

    • Optional opts: { skip?: boolean; pauseOnUserScroll?: boolean; initialBehavior?: ScrollIntoViewOptions["behavior"]; behavior?: ScrollIntoViewOptions["behavior"]; block?: ScrollIntoViewOptions["block"]; inline?: ScrollIntoViewOptions["inline"] }
      • Optional skip?: boolean

        set to true to skip auto scrolling altogether

      • Optional pauseOnUserScroll?: boolean

        set to true to stop autoscrolling when the user manually scrolls

      • Optional initialBehavior?: ScrollIntoViewOptions["behavior"]

        see https://developer.mozilla.org/en-US/docs/Web/API/Element/scrollIntoView

      • Optional behavior?: ScrollIntoViewOptions["behavior"]

        see https://developer.mozilla.org/en-US/docs/Web/API/Element/scrollIntoView

      • Optional block?: ScrollIntoViewOptions["block"]

        see https://developer.mozilla.org/en-US/docs/Web/API/Element/scrollIntoView

      • Optional inline?: ScrollIntoViewOptions["inline"]

        see https://developer.mozilla.org/en-US/docs/Web/API/Element/scrollIntoView

    Returns [RefCallback<E>, RefCallback<T>]

    A React ref object that should be set on the scroll container, and another ref that should be the target to scroll into view.
